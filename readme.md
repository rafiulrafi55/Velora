
# Wedding Guest Management System

## Overview
A web application for managing wedding guest lists, RSVPs, and event details.

## Features
- Guest list management
- RSVP tracking
- Seating arrangements
- Dietary preferences
- Budget management

## Status
🚧 **In Development**

## Getting Started
```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install

# Run the application
npm start
```

## Tech Stack
- Frontend: HTML
- Backend: Java script and php
- Database: mySQL
